Step to Setup the PropertyAnalysis Program

1. Download Folder 'Individual Project Property Analysis'.
2. Download the latest version of node.js
	https://nodejs.org/en/
2. Open a File explorer and navigate to 'Individual Project Property Analysis' Directory.
3. Open command prompt within the Directory.
	( Command Prompt can be open by typing 'cmd' in the address bar )
	( This will open Command Prompt in the location of the Directory )
4. Type 'npm -v' to verify node.js installation
5. Type 'npm install -g nodemon' to install nodemon

Steps to Run the PropertyAnalysis Program

1. Open command prompt within the 'Individual Project Property Analysis' Directory.
2. Type 'npm start'
3. Command Prompt will show the following:
	> individual-project-property-analysis@1.0.0 start
	> nodemon index.js

	[nodemon] 2.0.16
	[nodemon] to restart at any time, enter `rs`
	[nodemon] watching path(s): *.*
	[nodemon] watching extensions: js,mjs,json
	[nodemon] starting `node index.js`
	Server running on PORT 8000
4. Backend Server is now running locally.
5. Open home.html file within the 'Individual Project Property Analysis' Directory.
